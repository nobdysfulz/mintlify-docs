# Scorecards

Scorecards give team leaders a month-by-month view of every agent's lead generation activity. It is one of the most powerful tools for weekly accountability conversations.

## What the Scorecard Shows

Each agent has a scorecard grid with months across the top and activity metrics down the left side. Every cell shows that agent's actual number for that metric in that month.

### Metrics Tracked on Scorecards

| Metric | Description |
|--------|-------------|
| **Listing Appts Set** | Listing consultations scheduled |
| **Listing Appts Held** | Listing consultations that took place |
| **Listings Signed** | Signed listing agreements |
| **Buyer Appts Set** | Buyer consultations scheduled |
| **Buyer Appts Held** | Buyer consultations that took place |
| **Buyers Signed** | Signed buyer representation agreements |
| **Written Offers** | Offers submitted |
| **Contacts** | Prospecting contacts made |
| **Custom Metrics** | Any custom lead gen fields your team uses |

## Goal/Mo Column

The **Goal/Mo** column shows each agent's monthly target for that metric. This makes it easy to see at a glance whether an agent's month was above or below expectation.

## Color Coding

- **Green** -- Met or exceeded the monthly goal
- **Yellow** -- Close to goal but slightly under
- **Red** -- Significantly below goal
- **Current month** -- Highlighted to make the live month easy to spot

## YTD Total Column

The rightmost column shows each metric's year-to-date total.

## Using Scorecards for Coaching

Look for patterns, not just single months. If an agent has three red months in a row on Contacts, that is a prospecting habit conversation. If Listing Appts Held is red while Listing Appts Set is green, focus on the quality of appointments and pre-qualification.
