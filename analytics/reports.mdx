# Reports

The Reports page lets you generate formatted summaries of your team's data that are ready to print or save as a PDF.

## Available Report Types

### Financial Report
Monthly GCI, cost of sales, gross profit, net profit, and expense breakdown by category. Best for monthly financial reviews and broker conversations.

### Pipeline Report
All transactions grouped by status with total count and GCI value for each group. Best for pipeline reviews and production forecasting.

### Leaderboard Report
Ranked list of all agents showing GCI, units closed, and average sale price. Best for team meetings and recognition.

### Team Overview Report
Summary of all agents with key metrics and team-level totals. Best for franchise reporting and leadership presentations.

### Goals Report
Annual goals versus YTD actuals for each agent with pace indicators. Best for quarterly reviews and coaching conversations.

### Trend Report
Monthly production data in table format showing team totals and per-agent monthly breakdown. Best for year-over-year comparison.

## Generating a Report

1. Navigate to **Reports** in the sidebar
2. Select the report type you need
3. Choose any available filters (date range, agent, office)
4. The report renders on screen

## Printing and Saving as PDF

Click **Print** to open your browser's print dialog. From there you can print directly or save as a PDF using the "Save as PDF" option. Reports are formatted for clean printing with no sidebar or navigation elements included.
