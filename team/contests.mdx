# Contests

Contests let team leaders create time-bound competitions that reward agents for specific activities.

## Creating a Contest

1. Navigate to **Contest** in the sidebar
2. Click **Create Contest**
3. Fill in the contest details

### Contest Fields

| Field | Description |
|-------|-------------|
| **Name** | A memorable name for the contest |
| **Start Date** | When the contest begins |
| **End Date** | When the contest ends |
| **Description** | What the contest is about and what agents are competing for |

### Point Values

Assign point values to any combination of these activities:

| Activity | Description |
|----------|-------------|
| **Contacts** | Points per prospecting contact made |
| **Appts Set** | Points per appointment scheduled |
| **Appts Held** | Points per appointment that took place |
| **Written Offers** | Points per offer written |
| **Listings Taken** | Points per signed listing agreement |
| **Buyers Taken** | Points per signed buyer representation agreement |
| **Sold Units** | Points per closed transaction |

## Viewing Contest Results

- **Podium** -- Top 3 agents displayed with their point totals
- **Full Leaderboard** -- All participating agents ranked by points

Agents earn points automatically as they log daily lead gen activity and close transactions. No manual scoring needed.

## Active vs. Past Contests

The Contest page separates current and upcoming contests from completed ones.

## Contest Tips

- Keep contests short (1 to 4 weeks) to maintain urgency
- Weight activities that move the needle most for your team's current goals
- Announce the prize before the contest starts
- Review results in team meetings to keep energy high
