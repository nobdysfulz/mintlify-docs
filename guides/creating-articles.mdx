Knowledgebase gives you a powerful editor for writing help content that looks great and stays easy to maintain. Whether you prefer clicking toolbar buttons or writing raw Markdown, you can craft polished articles, embed media, and publish them to your customers in just a few steps.

## Writing your first article

<Steps>
  <Step title="Navigate to Articles → New Article">
    From your Knowledgebase dashboard, click **Articles** in the left sidebar, then select **New Article** in the top-right corner. A blank editor opens immediately — no configuration needed before you start writing.
  </Step>
  <Step title="Add a title and body">
    Click the **Title** field at the top of the editor and type a clear, descriptive name for your article. Then click into the body area below and start writing. The body supports full rich-text formatting as well as Markdown mode (see below).
  </Step>
  <Step title="Format with the toolbar">
    Use the floating toolbar to apply formatting as you write or after selecting text:
    - **Bold** (`B`) and *Italic* (`I`) for emphasis
    - **Headings** (H1–H3) to create scannable sections
    - **Ordered and unordered lists** for step-by-step instructions or feature lists
    - **Inline code** and **code blocks** for technical content
    - **Blockquotes** to highlight important callouts
    - **Horizontal rules** to separate major sections

    Select any text to reveal the inline toolbar, or use the `/` slash command to insert a block element at the cursor position.
  </Step>
  <Step title="Add tags and an SEO description">
    Scroll down to the **Article Settings** panel on the right side of the editor. Enter one or more **tags** to help readers find related content, then write a short **SEO description** (150–160 characters recommended) that appears in search engine results and in your help center's article preview cards.
  </Step>
  <Step title="Save as draft or publish">
    When you are ready, choose one of two actions in the top-right toolbar:
    - **Save Draft** — stores your work privately; customers cannot see the article yet.
    - **Publish** — makes the article immediately visible to anyone with access to your help center.

    You can switch an article back to draft at any time from its settings panel.
  </Step>
</Steps>

## Using Markdown mode

If you prefer writing in plain text, toggle Markdown mode by clicking the **`MD`** icon in the top-right corner of the editor. The editor switches to a monospace view where you write standard Markdown syntax.

Supported syntax includes:

| Element | Markdown syntax |
|---|---|
| Headings | `# H1`, `## H2`, `### H3` |
| Bold | `**bold**` |
| Italic | `*italic*` |
| Inline code | `` `code` `` |
| Code block | ```` ```language ```` |
| Unordered list | `- item` |
| Ordered list | `1. item` |
| Blockquote | `> quote` |
| Link | `[text](url)` |
| Image | `![alt](url)` |
| Horizontal rule | `---` |

Toggle back to rich-text mode at any time — Knowledgebase converts your Markdown to formatted content automatically.

## Adding media

### Images

Drag and drop an image file directly into the editor body to upload it instantly. Alternatively, click the **Image** block via the `/` slash command and paste a public URL to embed a hosted image. You can resize images and add alt text by clicking the image after it is inserted.

### Videos

To embed a video, use the `/` slash command and select **Video Embed**. Paste a YouTube, Vimeo, or Loom URL into the input field — Knowledgebase renders a playable embed inline in your article.

### Code snippets

Insert a code block with the `/` slash command → **Code Block**, or type three backticks (` ``` `) followed by the language name (e.g., `javascript`, `python`, `bash`). Code blocks include automatic syntax highlighting and a one-click copy button for readers.

## Article templates

If you write articles that follow a consistent structure — such as release notes, FAQs, or troubleshooting guides — save time with templates.

To **save an article as a template**:
1. Open the article you want to reuse.
2. Click the **⋯ More** menu in the top-right corner.
3. Select **Save as Template** and give the template a name.

To **create a new article from a template**:
1. Click **Articles → New Article**.
2. Select **Start from Template** in the new article dialog.
3. Choose your template from the list — the editor pre-fills with its structure and placeholder content.

Templates are private to your workspace and are not visible to customers.

<Tip>
  Speed up your editing with keyboard shortcuts. Use **Ctrl+B** (Windows) or **⌘+B** (Mac) for bold, **Ctrl+I** / **⌘+I** for italic, **Ctrl+K** / **⌘+K** to insert a link, and **Ctrl+Z** / **⌘+Z** to undo. Press `/` anywhere in the body to open the full block-insert menu without reaching for your mouse.
</Tip>

<Note>
  Knowledgebase autosaves your work every 30 seconds. You will see a **Saved** indicator appear briefly in the toolbar whenever a save occurs. Even if you close the tab unexpectedly, your most recent changes are preserved as a draft.
</Note>
